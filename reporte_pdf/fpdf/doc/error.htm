Error

Error(string msg)

Descripción

éste método es llamado automáticamente in caso de un error fatal. éste método simplemente devuelve una excepción con el mensaje correspondiente.
Una clase puede heredar éste método para personalizar el tratamiento del error. Pero el método nunca debe devolver, en tal caso, el resultado del documento puede ser erróneo.

Parámetros

msg
El mensaje de error.
